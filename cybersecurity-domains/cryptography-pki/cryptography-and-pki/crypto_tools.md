# Cryptography Ethical Hacking Tools

The following list includes some of the most popular tools to test crypto implementations.

| Name | Description |
|------|-------------|
| [aespipe](https://en.kali.tools/all/?tool=15) | Reads data from stdin and outputs encrypted or decrypted results to stdout. |
| [argon2](https://en.kali.tools/all/?tool=60) | The password hash Argon2, winner of PHC. |
| [armor](https://en.kali.tools/all/?tool=2114) | A simple Bash script designed to create encrypted macOS payloads capable of evading antivirus scanners. |
| [athena-ssl-scanner](https://en.kali.tools/all/?tool=75) | A SSL cipher scanner that checks all cipher codes. It can identify about 150 different ciphers. |
| [auto-xor-decryptor](https://en.kali.tools/all/?tool=78) | Automatic XOR decryptor tool. |
| [bletchley](https://en.kali.tools/all/?tool=136) | A collection of practical application cryptanalysis tools. |
| [brute12](https://en.kali.tools/all/?tool=162) | A tool designed for auditing the cryptography container security in PKCS12 format. |
| [bruteforce-luks](https://en.kali.tools/all/?tool=2267) | Try to find the password of a LUKS encrypted volume. |
| [bruteforce-salted-openssl](https://en.kali.tools/all/?tool=2268) | Try to find the password of a file that was encrypted with the 'openssl' command. |
| [bruteforce-wallet](https://en.kali.tools/all/?tool=163) | Try to find the password of an encrypted Peercoin (or Bitcoin, Litecoin, etc.) wallet file. |
| [check-weak-dh-ssh](https://en.kali.tools/all/?tool=203) | Debian OpenSSL weak client Diffie-Hellman Exchange checker. |
| [chrome-decode](https://en.kali.tools/all/?tool=216) | Chrome web browser decoder tool that demonstrates recovering passwords. |
| [cipherscan](https://en.kali.tools/all/?tool=222) | A very simple way to find out which SSL ciphersuites are supported by a target. |
| [ciphertest](https://en.kali.tools/all/?tool=223) | A better SSL cipher checker using gnutls. |
| [ciphr](https://en.kali.tools/all/?tool=224) | A CLI tool for encoding, decoding, encryption, decryption, and hashing streams of data. |

| [cisco7crack](https://en.kali.tools/all/?tool=235) | Crypt and decrypt the cisco enable 7 passwords. |

| [dagon](https://en.kali.tools/all/?tool=1791) | Advanced Hash Manipulation. |

| [decodify](https://en.kali.tools/all/?tool=1963) | Tool that can detect and decode encoded strings, recursively. |
| [deen](https://en.kali.tools/all/?tool=1720) | Generic data encoding/decoding application built with PyQt5. |
| [demiguise](https://en.kali.tools/all/?tool=2240) | HTA encryption tool for RedTeams. |
