# Social Engineering

Social engineering is the practice of manipulating people into revealing sensitive information, performing actions, or downloading malware by using psychological tactics. It is a type of attack that is often used to gain access to systems or networks.

## Tools

- [Social Engineering Toolkit (SET)](https://github.com/trustedsec/social-engineer-toolkit)
- [PyPhishing Toolkit](https://github.com/redteamsecurity/PyPhishing)
- [Phishing on Twitter - POT](https://www.kitploit.com/2018/02/pot-phishing-on-twitter.html)
- [Evilginx2](https://github.com/kgretzky/evilginx2) - MiTM framework used for phishing login credentials along with session cookies and can help bypass 2-factor authentication.
